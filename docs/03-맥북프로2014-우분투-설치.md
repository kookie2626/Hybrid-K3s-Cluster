# 맥북프로 2014에 우분투 설치하기

## 배경

쿠버네티스(K3s)를 배운 후, 단일 서버에 머무르지 않고 여러 대의 장치를 활용해 실제 멀티노드 클러스터를 구성해보고 싶었습니다.
추가 서버를 구매하는 대신, 집에 있던 **맥북프로 2014(Intel i5)** 를 활용하기로 했습니다.

인텔 맥이어서 x86-64 아키텍처를 지원하므로 Ubuntu가 설치 가능하다는 것을 확인했습니다.
맥OS는 2021년부터 해당 기기에 대한 보안 업데이트 지원이 중단된 상태였기 때문에, 맥OS를 완전히 삭제하고 **Ubuntu 22.04 LTS**를 설치했습니다.

> 💡 10년이 넘은 노트북을 K3s 워커 노드로 재활용함으로써 추가 비용 없이 클러스터를 확장할 수 있었습니다.

## Ubuntu 설치 준비

### 1. Ubuntu 22.04 LTS ISO 다운로드

[Ubuntu 공식 사이트](https://ubuntu.com/download/desktop)에서 Ubuntu 22.04 LTS Desktop ISO를 다운로드합니다.

> 서버 용도이므로 Ubuntu Server ISO를 사용해도 무방합니다.

### 2. 부팅 USB 만들기

```bash
# macOS/Linux에서 dd 명령어로 부팅 USB 제작 (디바이스명은 환경에 따라 다름)
sudo dd if=ubuntu-22.04-live-server-amd64.iso of=/dev/sdX bs=4M status=progress
```

Windows에서는 [Rufus](https://rufus.ie/)를 사용해 부팅 USB를 만들 수 있습니다.

## Ubuntu 설치 과정

### 1. 맥OS 디스크 완전 초기화

맥북프로에서 **Command + R** 을 누른 채 전원을 켜 **macOS 복구 모드**로 진입합니다.

```
복구 모드 진입 → 디스크 유틸리티 → Macintosh HD 선택 → 지우기
```

디스크를 지운 후 복구 모드를 종료합니다.

### 2. Ubuntu USB로 부팅

부팅 USB를 꽂고 전원을 켤 때 **Option(⌥)** 키를 누르고 있으면 부팅 디스크 선택 화면이 나타납니다.
USB 드라이브(EFI Boot)를 선택합니다.

### 3. Ubuntu 설치 마법사 진행

- 언어: English (권장) 또는 한국어
- 키보드: English US
- 설치 유형: **Erase disk and install Ubuntu** 선택 (맥OS가 이미 삭제된 상태)
- 사용자명 / 호스트명 / 비밀번호 설정
- **OpenSSH server 설치 체크** (이후 원격 접속을 위해 필수)

설치 완료 후 USB를 제거하고 재부팅합니다.

## 하드웨어 호환성 이슈 해결

인텔 맥에 Ubuntu를 설치하면 일부 하드웨어가 즉시 동작하지 않는 경우가 있습니다.

### Wi-Fi 드라이버 (Broadcom BCM4360)

맥북프로 2014의 Wi-Fi 칩(Broadcom BCM4360)은 Ubuntu 기본 커널에서 지원되지 않아 별도 설치가 필요합니다.

> ⚠️ **트러블슈팅 기록**: Ubuntu 22.04 기본 커널(5.15)에서는 `bcmwl-kernel-source` 설치 후에도 Wi-Fi가 인식되지 않는 문제가 발생했습니다. 드라이버 설치를 여러 차례 시도하였으나 계속 실패하였고, **커널을 6.8로 업그레이드한 후에야 드라이버가 정상 동작**했습니다.

#### Step 1: 커널 6.8로 업그레이드 (HWE)

Ubuntu 22.04 LTS의 HWE(Hardware Enablement) 스택을 통해 커널 6.8을 설치합니다.
유선 인터넷(이더넷)으로 연결한 뒤 실행하세요.

```bash
sudo apt update
sudo apt install -y linux-generic-hwe-22.04
sudo reboot
```

재부팅 후 커널 버전을 확인합니다:

```bash
uname -r
# 예: 6.8.0-xx-generic
```

#### Step 2: Broadcom Wi-Fi 드라이버 설치

커널 6.8 부팅 후, 드라이버를 설치합니다:

```bash
sudo apt update
sudo apt install -y bcmwl-kernel-source
sudo modprobe wl
```

설치 후 재부팅하면 Wi-Fi가 정상적으로 인식됩니다.

### 키보드 펑션키(Fn) 설정

맥 키보드는 기본적으로 F1~F12가 미디어 키로 동작합니다. 일반 F-key로 동작하도록 변경합니다.

```bash
echo "options hid_apple fnmode=2" | sudo tee /etc/modprobe.d/hid_apple.conf
sudo update-initramfs -u -k all
```

재부팅 후 F1~F12가 표준 기능 키로 동작합니다.

## 설치 후 기본 셋업

```bash
# 패키지 업데이트
sudo apt update && sudo apt upgrade -y

# 기본 패키지 설치
sudo apt install -y curl wget git vim htop btop net-tools
```

SSH 접속 설정, 고정 IP 설정은 [N100 서버 셋업 문서](01-n100-서버-셋업.md)와 동일한 방법으로 진행합니다.

자동화 스크립트를 사용하려면 아래를 참고하세요:

```bash
# 스크립트 실행 (저장소 루트에서)
chmod +x scripts/setup/mbp-2014-ubuntu-setup.sh
sudo scripts/setup/mbp-2014-ubuntu-setup.sh
```
